# Derailed

You're probably looking for [derailed benchmarks](https://github.com/schneems/derailed_benchmarks).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'derailed'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install derailed
